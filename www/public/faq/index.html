Frequently Asked Questions

Last updated: Jan 2026
What is 4share?
4share is a free peer-to-peer file transfer tool that works directly in your web browser. It uses WebRTC technology to create a direct, encrypted connection between two devices, allowing files to transfer without ever touching a server.
How does 4share work?
Open 4share on the sending device, select files, and get a 6-digit code. Share that code with the receiver (via text, QR code, or verbally). The receiver enters the code on their device, and files transfer directly between both devices using a peer-to-peer WebRTC connection.
Is 4share free?
Yes, completely free. There is no premium tier, no subscription, no ads, and no hidden costs. The entire product is free forever.
Do I need to create an account?
No. 4share requires zero signup. Open the website, and start transferring files immediately. No email, no password, no personal information required.
Is there a file size limit?
No. 4share has no file size limits. You can send files of any size — from a few kilobytes to hundreds of gigabytes.
Are my files stored on a server?
No. Files transfer directly between devices using peer-to-peer technology. They never touch any server, cloud storage, or third-party infrastructure.
How is 4share different from WeTransfer?
WeTransfer uploads files to their servers, then the recipient downloads from there. 4share transfers files directly between devices (P2P). This makes 4share faster, more private (no server storage), and unlimited (no file size caps).
How is 4share different from Send-Anywhere?
Send-Anywhere may route transfers through its servers and displays ads on the free tier. 4share is pure P2P (no server storage), has no ads, and no file size limits.
Does 4share work on mobile?
Yes. 4share works in any modern browser on any device — iPhone, Android, tablets, laptops, and desktops. No app download needed.
How do I send files?
Go to www.4share.work, click "Send Files," select your files, and you'll get a 6-digit code. Share that code with the recipient. They enter it on their device to receive the files.
How do I receive files?
Go to www.4share.work, click "Receive Files," and enter the 6-digit code the sender gave you. The files will transfer directly to your device.
What is a 6-digit code?
The 6-digit code is a unique identifier that connects the sender and receiver. When the sender generates a code, it creates a temporary room. When the receiver enters the same code, they join that room and the transfer begins.
Is my transfer encrypted?
Yes. All 4share transfers use WebRTC with DTLS (Datagram Transport Layer Security) encryption. This is the same encryption used in secure video calls and banking applications.
Can I send multiple files at once?
Yes. Select multiple files before generating the code. All selected files will be transferred together in sequence.
How fast is 4share?
Speed depends on your internet connection. On the same WiFi network, transfers can exceed 100 MB/s. Over the internet, speed is limited by the slower of the two connections. P2P is generally faster than cloud-based services because there's no server bottleneck.
What browsers are supported?
4share works in all modern browsers: Chrome, Firefox, Safari, Edge, and their mobile versions. Chrome and Edge provide the best WebRTC performance.
Can I send files to someone in a different country?
Yes. 4share works over the internet, so you can send files to anyone anywhere in the world. Just share the 6-digit code via any messaging app.
What happens if the connection drops?
4share uses chunked transfer. If the connection drops, the transfer pauses. When the connection is restored, you can restart the transfer. The receiver gets whatever was successfully transferred.
Is 4share open source?
Yes. The code is available on GitHub. You can view it, report issues, and contribute at github.com/falsafi2-0/shareit.
