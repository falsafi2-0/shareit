How 4share Works: Send Files in 5 Simple Steps

Published: Jan 2026 Category: Tutorial 5 min read

4share lets you transfer files directly between devices using peer-to-peer technology. No signup, no file size limits, no ads. Here's exactly how to use it.

Step 1: Select Your Files

1

Open 4share and Choose Files

Go to www.4share.work on the sending device. Click "Send Files" to get started. You'll see a file picker where you can:

  • Click to browse — open your file browser and select one or more files
  • Drag and drop — drag files directly from your desktop or file manager onto the page
  • Select multiple files — hold Ctrl (Windows) or Cmd (Mac) to select several files at once

4share supports any file type — documents, photos, videos, music, archives, anything. There is no file size limit.

After selecting your files, you'll see a summary showing the number of files and total size. You can add more files or remove any you don't want to send.

Step 2: Share the Code

2

Get Your 6-Digit Transfer Code

After clicking "Generate Code", 4share creates a unique 6-digit code for your transfer session. This code is how the receiver connects to your device.

You'll see:

  • The 6-digit code — displayed in large numbers for easy reading
  • A QR code — scan this with the receiver's phone camera for instant connection
  • A shareable link — copy and send this link via any messaging app

Share the code with the receiver using whatever method is convenient:

  • Read it out loud if they're nearby
  • Text it via WhatsApp, iMessage, SMS, etc.
  • Have them scan the QR code with their phone camera
  • Send the shareable link

The code expires after 10 minutes for security. If it expires, simply generate a new one.

Step 3: Receiver Enters the Code

3

Connect the Receiving Device

On the receiving device, open www.4share.work in any browser. Click "Receive Files".

You'll see a 6-digit code input screen. Enter the code from the sender:

  • Type the code — enter each digit in the input boxes
  • Paste the code — if you received the code as text, paste it and all digits fill in automatically
  • Click "Receive Files" — or it auto-submits when all 6 digits are entered

If you scanned the QR code, the code is entered automatically and the transfer starts immediately.

Step 4: Transfer Starts Automatically

4

Files Transfer Directly Between Devices

Once the receiver enters the code, 4share establishes a direct peer-to-peer connection between the two devices using WebRTC.

What happens behind the scenes:

  • The devices find each other through Cloudflare's signaling server
  • A secure, encrypted connection is established directly between browsers
  • Files begin streaming from sender to receiver
  • Files never touch any server — they go straight from device to device

On both screens, you'll see a progress bar showing:

  • Transfer percentage
  • Transfer speed (KB/s or MB/s)
  • Connection type (Direct P2P)

The transfer speed depends on your internet connection. On the same WiFi network, speeds can exceed 100 MB/s.

Step 5: Done!

5

Files Saved to Downloads

When the transfer completes, the files are automatically saved to the receiving device's Downloads folder.

You'll see a confirmation screen showing:

  • "Transfer complete!" message
  • Total files received
  • Option to send or receive more files

That's it — no account creation, no email required, no waiting for uploads to a server. The entire transfer happened directly between your devices.

Tips for Best Results

  • Same WiFi is fastest — if both devices are on the same network, the transfer uses local bandwidth and is much faster
  • Keep the tab open — don't close the browser tab during transfer
  • Use Chrome or Edge — these browsers have the best WebRTC support
  • Check your firewall — some corporate firewalls may block P2P connections
  • Large files work great — 4share handles files of any size without issues

Frequently Asked Questions

Do I need to install an app?
No. 4share works entirely in your web browser. No app download, no installation, no setup required.
Does it work on mobile?
Yes. 4share works on any device with a web browser — iPhone, Android, tablets, laptops, and desktops.
What if the code expires?
Codes expire after 10 minutes for security. If yours expires, simply go back and generate a new code. It only takes a second.
Can I send multiple files at once?
Yes. Select multiple files before generating the code. All selected files will be transferred together.
Is there a file size limit?
No. 4share has no file size limits. You can send files of any size — from a few kilobytes to hundreds of gigabytes.
How fast is the transfer?
Speed depends on your internet connection. On the same WiFi network, transfers can exceed 100 MB/s. Over the internet, speed is limited by the slower connection.
Are my files stored on a server?
No. Files transfer directly between devices using peer-to-peer technology. They never touch any server or cloud storage.
What browsers are supported?
4share works in all modern browsers — Chrome, Firefox, Safari, Edge, and their mobile versions. Chrome and Edge provide the best performance.
